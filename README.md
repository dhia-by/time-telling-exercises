# Time-Telling Exercises for Kids

An interactive website with 20 time-telling exercises featuring real analog clock images for children to practice reading time.

## Features

- **20 Interactive Exercises**: Each exercise displays a real analog clock image
- **User-Friendly Interface**: Clean, colorful design optimized for children
- **Answer Key**: Toggle to reveal answers for self-checking
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **No Dependencies**: Pure HTML, CSS, and JavaScript

## Getting Started

### Local Development

1. Navigate to the project directory:
   ```bash
   cd /home/ubuntu/time_telling_exercises
   ```

2. Start a local server:
   ```bash
   python3 -m http.server 8000 --directory public
   ```

3. Open your browser and visit:
   ```
   http://localhost:8000
   ```

## How to Use

1. Look at each analog clock face
2. Write the time you see in the input field
3. Click "Show Answer Key" to check your answers
4. Practice regularly to improve your time-reading skills!

## Content

The website includes 20 exercises with times ranging from 1:00 to 8:45, covering:
- Hours
- Half-hours
- Quarter-hours
- Various minute intervals
